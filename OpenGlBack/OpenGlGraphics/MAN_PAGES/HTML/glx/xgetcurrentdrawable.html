NAME
	  glXGetCurrentDrawable	- return the current drawable


     C SPECIFICATION
	  GLXDrawable glXGetCurrentDrawable( void )


     DESCRIPTION
	  glXGetCurrentDrawable	returns	the current drawable, as
	  specified by glXMakeCurrent.	If there is no current
	  drawable, None is returned.

	  glXGetCurrentDrawable	returns	client-side information.  It
	  does not make	a round	trip to	the server.

     SEE ALSO
	  glXCreateGLXPixmap, glXMakeCurrent
