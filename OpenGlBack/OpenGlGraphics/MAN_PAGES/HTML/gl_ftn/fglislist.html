NAME
	  fglIsList - determine	if a name corresponds to a display-
	  list


     FORTRAN SPECIFICATION
	  LOGICAL*1 fglIsList( INTEGER*4 list )


     PARAMETERS
	  list	Specifies a potential display-list name.

     DESCRIPTION
	  fglIsList returns GL_TRUE if list is the name	of a display
	  list and returns GL_FALSE otherwise.

     ERRORS
	  GL_INVALID_OPERATION is generated if fglIsList is executed
	  between the execution	of fglBegin and	the corresponding
	  execution of fglEnd.

	  fglNewList
