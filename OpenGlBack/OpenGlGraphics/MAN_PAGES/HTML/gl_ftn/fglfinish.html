NAME
	  fglFinish - block until all GL execution is complete


     FORTRAN SPECIFICATION
	  SUBROUTINE fglFinish(	)


     DESCRIPTION
	  fglFinish does not return until the effects of all
	  previously called GL commands	are complete.  Such effects
	  include all changes to GL state, all changes to connection
	  state, and all changes to the	frame buffer contents.

     NOTES
	  fglFinish requires a round trip to the server.

     ERRORS
	  GL_INVALID_OPERATION is generated if fglFinish is executed
	  between the execution	of fglBegin and	the corresponding
	  execution of fglEnd.
