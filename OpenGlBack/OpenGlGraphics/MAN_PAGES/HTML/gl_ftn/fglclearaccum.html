NAME
	  fglClearAccum	- specify clear	values for the accumulation
	  buffer


     FORTRAN SPECIFICATION
	  SUBROUTINE fglClearAccum( REAL*4 red,
				    REAL*4 green,
				    REAL*4 blue,
				    REAL*4 alpha )


     PARAMETERS
	  red, green, blue, alpha
				Specify	the red, green,	blue, and
				alpha values used when the
				accumulation buffer is cleared.	 The
				initial	values are all 0.

     DESCRIPTION
	  fglClearAccum	specifies the red, green, blue,	and alpha
	  values used by fglClear to clear the accumulation buffer.

	  Values specified by fglClearAccum are	clamped	to the range
	  [-1,1].

     ERRORS
	  GL_INVALID_OPERATION is generated if fglClearAccum is
	  executed between the execution of fglBegin and the
	  corresponding	execution of fglEnd.

     ASSOCIATED	GETS
	  fglGet with argument GL_ACCUM_CLEAR_VALUE

     SEE ALSO
	  fglClear
