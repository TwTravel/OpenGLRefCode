NAME
	  fgluDeleteNurbsRenderer - destroy a NURBS object


     FORTRAN SPECIFICATION
	  SUBROUTINE fgluDeleteNurbsRenderer( CHARACTER*8 nurb )


     PARAMETERS
	  nurb	Specifies the NURBS object to be destroyed.

     DESCRIPTION
	  fgluDeleteNurbsRenderer destroys the NURBS object (which was
	  created with fgluNewNurbsRenderer) and frees any memory it
	  uses.	 Once fgluDeleteNurbsRenderer has been called, nurb
	  cannot be used again.

     SEE ALSO
	  fgluNewNurbsRenderer
