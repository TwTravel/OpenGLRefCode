NAME
	  fgluNurbsCallbackDataEXT - set a user	data pointer


     FORTRAN SPECIFICATION
	  SUBROUTINE fgluNurbsCallbackDataEXT( CHARACTER*8 nurb,
					       CHARACTER*8 userData )


     PARAMETERS
	  nurb	    Specifies the NURBS	object (created	with
		    fgluNewNurbsRenderer).

	  userData  Specifies a	pointer	to the user's data.

     DESCRIPTION
	  fgluNurbsCallbackDataEXT is used to pass a pointer to	the
	  application's	data to	NURBS tessellator. A copy of this
	  pointer will be passed by the	tessellator in the NURBS
	  callback functions (set by fgluNurbsCallback).
