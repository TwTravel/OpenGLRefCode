NAME
	  fgluNewNurbsRenderer - create	a NURBS	object


     FORTRAN SPECIFICATION
	  CHARACTER*8 fgluNewNurbsRenderer( )


     DESCRIPTION
	  fgluNewNurbsRenderer creates and returns a pointer to	a new
	  NURBS	object.	 This object must be referred to when calling
	  NURBS	rendering and control functions. A return value	of 0
	  means	that there is not enough memory	to allocate the
	  object.
