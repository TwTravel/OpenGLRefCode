插件用结构说明
 
 
本节将描述图像读写插件模块内部使用的几个结构（struct），弄懂这些结构的含义及使用 方法对编写新的图像读写插件是十分必要的。请选择查阅。


            1、IRWP_INFO          - 插件信息表
            2、IRW_SAVE_PARA_DESC   - 保存功能参数设定描述
            3、IRW_SAVE_DESC      - 保存功能描述
            4、SAVESTR              - 保存图像时用于设定参数的结构
            5、AUTHOR_INFO        - 开发者信息
            6、IRWP_DESC_INFO       - 插件能处理的图像文件类型信息


上面的结构定义于iseeexplorer\include目录下的iseeirw.h文件中。有关 INFOSTR 和 SUBIMGBLOCK 结构的含义请参见接口用结构说明一章。
